Usage: firebase apphosting:secrets:set [options] <secretName>

create or update a secret for use in Firebase App Hosting

Options:
  -l, --location <location>  optional location to retrict secret replication
  -f, --force                Automatically create a secret, grant permissions,
                             and add to YAML.
  --data-file <dataFile>     File path from which to read secret data. Set to
                             "-" to read the secret data from stdin.
  -h, --help                 display help for command
